# 行为准则

## 基本标准

本项目欢迎所有希望让 AI 辅助 WordPress 建站更安全、更稳定、更有用的贡献者。

鼓励的行为：

- 尊重他人，表达具体。
- 针对流程、代码和文档提出批评，不做人身攻击。
- 把用户安全、隐私和可维护性放在第一位。
- 不分享凭据、客户数据或私有站点资料。
- 报告风险、限制或验证失败时保持清晰。

不接受的行为：

- 骚扰、仇恨言论、威胁或人身攻击。
- 发布密钥、客户隐私数据或可利用漏洞细节。
- 鼓励危险的线上站点修改、凭据泄露或虚假宣传。
- 反复无视维护者关于范围、安全或文档质量的要求。

## 执行

维护者可以编辑、关闭、隐藏或删除违反本准则的 issue、PR、评论或 release。严重或重复违规者可能被禁止参与项目。

## 报告

行为问题请尽量私下联系维护者。安全漏洞请按照 `SECURITY.zh-CN.md` 处理。
