# Support

## Before Asking for Help

Please read:

- `README.md`
- `INSTALL.md`
- `docs/DOCUMENTATION_INDEX.md`
- `docs/RESUME_PROTOCOL.md`
- `docs/AI_TOOL_COMPATIBILITY.md`

For WordPress live-site work, include verification evidence. Do not include passwords, application passwords, tokens, cookies, private customer data, or order exports.

## Good Support Requests

Include:

- AI tool being used: Codex, Claude, Trae, Cursor, OpenHands, Aider, or other.
- Service mode: `new`, `old_rebuild`, or `existing_seo_optimization`.
- WordPress stack: theme, Elementor, WooCommerce, Rank Math, Code Snippets versions if known.
- What you expected.
- What happened.
- Relevant logs with secrets removed.
- Which helper script failed, if any.

## Not Covered

This project does not provide emergency production support, legal advice, medical/compliance advice, payment-processing certification, or guaranteed SEO rankings.

Security issues should follow `SECURITY.md`, not public support threads.
